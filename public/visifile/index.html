Yazz Pilot
KILL BUILD
BUILD WEB APPS FAST ON KUBERNETES. NO CI/CD NEEDED!
Yazz Pilot is a browser based IDE to build internal dashboards on Kubernetes or Docker. It can connect to Rest APIs, Postgres, and other stuff. JavaScript is used for scripting events and writing code.
